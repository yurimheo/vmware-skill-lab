# VMware Skill Lab

VKS L100 Foundation 학습 내용을 기반으로 만든 정적 웹 학습 사이트입니다.

## 목표

- VKS L100 스킬셋을 이론 퀴즈와 커맨드 입력 실습으로 점검합니다.
- 각 파트별 정답률을 계산하여 부족한 영역을 확인합니다.
- GitHub Pages에 바로 올릴 수 있도록 의존성 없는 정적 사이트로 구성합니다.

## 구성

| 파일 | 용도 |
| --- | --- |
| `index.html` | 앱 진입점 |
| `styles.css` | 화면 스타일 |
| `app.js` | 문제 렌더링, 채점, 진행률 저장 |
| `data/vks-l100.js` | VKS L100 문제 데이터 |

## 로컬 실행

정적 파일이므로 `index.html`을 브라우저에서 직접 열 수 있습니다.

로컬 서버로 확인하려면 아래 명령을 사용할 수 있습니다.

```bash
python3 -m http.server 4173
```

그 뒤 브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:4173
```

## GitHub Pages 배포

GitHub 저장소에 이 폴더 내용을 올린 뒤, Pages 설정에서 배포 브랜치와 폴더를 지정하면 됩니다.

초기 MVP는 외부 라이브러리 없이 동작하도록 만들었기 때문에 별도 빌드 과정이 필요하지 않습니다.
